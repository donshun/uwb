3D Chart Simulation
=======================

The assets provided in this directory give tools for drawing 3D Chart.

please use "Scenes/demo" to run the demo file.

In the scripts:
"cameraControl.cs": describes rotate/move/zoom of camera view
"chartControl.cs": draws 5 set of data. 1, solid line with filled area; 2, rainbow line; 3, dash line; 
                          4, bar gragh; 5, color line
"inputData.cs": read line data
"rateInputField.cs": add delegate of the axis rate inputfield
"selectView": select view button, fix axis rate button


have fun !
